# web-frontend

A multi-page online portfolio designed to showcase technical skills and academic projects. This project highlights the ability to build organized, interactive, and user-friendly web applications, with a focus on clean code and a professional presentation.

# Technical Implementation

Infrastructure: Built using semantic HTML5 to ensure accessibility and a hierarchical document structure optimized for search engines.

Design and Layout: Developed using custom CSS3, featuring responsive designs and a modern user interface without the need for external frameworks.

Navigation: A seamless, multi-page structure provides an intuitive user experience.
